# butler-vmobile
外推网-移动端vue项目

## package.json中关闭eslint规则的注解

```
"rules": {
  "no-param-reassign": 0, // 关闭给参数重新赋值引起的报错
  "consistent-return": 0, // 关闭箭头函数必须要有一个返回值的报错
  "linebreak-style": 0, // 关闭换行符报错
  "prefer-destructuring": 0, //关闭数组或对象不能强制解构,宁愿销毁的报错,
  "no-multi-assign": 0 // 关闭链式赋值的报错
}
```
